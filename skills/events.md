# Event Skills

## Event Types on the Platform

### By Frequency (most common first)
1. **Weddings** — highest value, longest planning cycle (6–18 months), most vendors per event (5–10)
2. **Birthday parties** — most common event type, shorter cycle (2–8 weeks), 1–3 vendors
3. **Corporate events** — team events, conferences, holiday parties, mixed budgets
4. **Baby showers / christenings** — growing category, 1–3 vendors
5. **Anniversaries** — milestone celebrations, similar vendor needs to weddings (smaller scale)
6. **Graduation / prom** — seasonal (May–July), mainly venue + DJ + photographer
7. **Festivals / community events** — larger scale, multiple vendor categories

### By Vendor Requirements
| Event Type | Typical Vendors Needed |
|-----------|----------------------|
| Wedding | Photographer, Caterer, Venue, Florist, DJ/Band, Videographer, Cake, Decorator |
| Birthday | Venue, Caterer, DJ, Decorator, Cake |
| Corporate | Venue, Caterer, Photographer, DJ/Band |
| Baby Shower | Venue, Decorator, Cake, Photographer |
| Anniversary | Venue, Caterer, Photographer, Florist, Cake |

## Seasonal Calendar

### Peak Seasons
| Season | Events | Marketing Focus |
|--------|--------|----------------|
| Jan–Feb | New Year planning | "Start planning your [year] wedding" |
| Mar–Apr | Wedding season begins | Vendor availability urgency |
| May–Jul | Weddings, proms, graduations, festivals | Peak demand, showcase reviews |
| Aug–Sep | Late summer weddings, back-to-school | "Last spots for autumn" |
| Oct–Nov | Halloween, corporate holiday planning | Corporate event marketing |
| Dec | Christmas parties, NYE | "Book your Christmas party vendors" |

### Off-Season Strategy (Jan–Mar)
- Vendor acquisition: easier to onboard vendors when they're less busy
- Content creation: publish planning guides for upcoming season
- Feature development: build new platform features before peak demand
- Email campaigns: "Wedding season is 3 months away — secure your vendors now"

## Event Planning Workflow (AI Planner)

### How the AI Planner Works
1. Customer inputs: event type, date, guest count, budget, theme/preferences
2. Claude AI generates a structured plan with:
   - **Budget breakdown** by vendor category (percentages and amounts)
   - **Vendor category recommendations** with priority levels
   - **Timeline/checklist** with tasks and deadlines
   - **Planning tips** specific to the event type
   - **Vendor suggestions** based on category and location
3. Customer can save, edit, or re-run the plan
4. Saved plans link to marketplace categories for easy vendor browsing

## Event Checklist Templates

### Wedding Checklist (12-month planning)
- 12 months: Set budget, book venue, hire photographer
- 9 months: Book caterer, florist, entertainment
- 6 months: Book videographer, cake maker, decorator
- 3 months: Final tastings, confirm all vendors, send invitations
- 1 month: Final payments, timeline confirmation, emergency contacts
- 1 week: Final vendor check-ins, seating plan, day-of timeline

### Birthday Party Checklist (6-week planning)
- 6 weeks: Set budget, choose theme, book venue
- 4 weeks: Book caterer/food, entertainment, send invitations
- 2 weeks: Order cake, confirm decorations, plan activities
- 1 week: Confirm all vendors, buy supplies, prep playlist
- Day before: Set up decorations, confirm delivery times

## Vendor Coordination Patterns

### Multi-Vendor Events
- Customers booking 3+ vendors for one event need coordination tools
- Event checklist tracks which vendors are booked vs still needed
- Messaging keeps vendor conversations separate but accessible
- Calendar shows all vendor bookings for one event in a single view

### Vendor-to-Vendor Awareness
- Vendors on the same event don't currently see each other
- Future consideration: shared event timeline visible to all booked vendors
- Vendor referrals: photographers recommend specific florists, caterers recommend venues

## Quality Signals

### What Makes a High-Quality Event Vendor on Event Nest
- Complete profile (100% completion score)
- Professional portfolio images (not phone snaps)
- Clear service packages with transparent pricing
- Quick response time to inquiries (tracked and displayed)
- Verified reviews with customer photos
- Active presence: updated availability, current promotions
