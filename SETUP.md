# SUPER SIMPLE SETUP - 3 STEPS

## Step 1: Extract This Folder
Put the "event-nest-SIMPLE" folder on your Desktop

## Step 2: Open Terminal & Run These 2 Commands

```bash
cd Desktop/event-nest-SIMPLE
npm install
```

Wait for it to finish (1-2 minutes)

## Step 3: Start It

```bash
npm run dev
```

Then go to: **http://localhost:3000**
